该项目用于学习，包含了状态管理库包含Zustand/Jotai/Valtio/React-Query实现，代码中包含了详细的注释用来学习实践。

## 使用

安装：

```
pnpm install
```

打包：

```
pnpm run build
```

运行单元测试：

```
pnpm run test
```

格式化

```
pnpm run prettier
```

eslint

```
pnpm run eslint
```
